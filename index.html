Frequently Asked Questions

Do the developers announce when there is a bug?

The developer team do announce when there is a bug/issue with the bot. We do our best to repair and improve the bot when this happens.

Can anyone use Bob the Bot?

Of course anyone can, Bob the Bot is a public bot so anyone can use it and anyone can add it to their own Discord server that follows Discord's ToS.

How can I contribute?

You can contribute on our GitHub repository by opening a pull request and sharing what you want to contribute.

How can I report an issue?

You can report a bug, issue, etc. By informing one of the developers via Discord or email [explodecode-help-business@outlook.com](mailto:explodecode-help-business@outlook.com) to contact the owner
